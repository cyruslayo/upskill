# Introduction to Algorithms and Machine Learning

Welcome to the interactive, from-scratch curriculum for algorithms and machine learning! 

Based on the highly rigorous pedagogy of **The Math Academy Way**, this curriculum takes you from basic variables and loops all the way to backpropagation and neuroevolution—all without relying on "black box" external machine learning libraries.

## Pedagogical Principles
1. **Coding from Scratch**: Build every algorithm (Gradient Descent, K-Means, Neural Networks) using base Python.
2. **Micro-Scaffolding**: Complex tasks are broken down into bite-sized code cells.
3. **Interleaved Practice**: Problem types are mixed to improve learning efficiency.
4. **Spaced Repetition**: Explicit sections dedicated to retrieving and practicing older concepts.
5. **Math Foundations**: Immediate references to targeted Algebra, Calculus, and Linear Algebra prerequisites.

## Project Structure
- `notebooks/`: Contains the 43 student-facing Jupyter Notebooks for algorithms/machine learning, plus 7 dedicated **Math Foundations** Notebooks.
- `solutions/`: Contains the completed, fully-coded versions for reference.
- `datasets/`: Toy datasets (CSV files) loaded in later regression and classification modules.
- `mathematics/`: Markdown textbooks acting as foundational math references.
- `learning_path.md`: The master curriculum map.

## Setup & Installation
1. Install Python 3.10+
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Launch Jupyter Notebook:
   ```bash
   jupyter notebook
   ```
4. Open the `notebooks/` directory and start with `01_introductory_coding.ipynb`!
